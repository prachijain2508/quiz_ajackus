## Quiz Management System - Plan

## Assumptions 
- No authetication required
- Only admin can create quizzes
- Quizzes are accessible for anyone
- Score based on correct answers only

## Scope
- Admin can create quizzes with multiple questions
- Question types are MCQs, True/False, Text

## Approach
- Monolith rails application
- Postgresql for database

## If more time
- Admin authentication
- Time-based quizzes
- Analytics dashboard
